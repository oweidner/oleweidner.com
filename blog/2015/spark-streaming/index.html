Configure Twitter API Access

For this example we’ll use the public tweet stream data that Twitter provides for development purposes. In order to access these streams you will need to set up API access and OAuth authentication for your Twitter account. Start at https://dev.twitter.com/apps and follow the instructions below.

Add a Name and a Description for your application. You can use any URL as Website. We don’t need callbacks, so we can leave Callback URL empty.

The next screen shows the basic settings of your new application. Click on the Keys and Access Tokens tab. At the bottom, click the Create my access token button.
